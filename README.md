Insertion Sort in JavaScript

Description

This project implements the Insertion Sort algorithm using JavaScript.

Insertion Sort is a simple sorting technique that works like sorting playing cards in your hand.
Each element is taken one by one and placed in its correct position in the already sorted part of the array.


Problem Statement

Sort an array using Insertion Sort by following these rules:

Work only with the first i-1 elements of the array
Pick element arr[i]
Insert it into its correct position in the sorted part of the array (from 0 to i-1)


Algorithm Idea

The array is divided into:
Sorted part (left side)
Unsorted part (right side)
Each element from the unsorted part is inserted into the correct position in the sorted part# Checkpoint-Sorting-and-searching-algorithms
